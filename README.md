# 🌍 World Environment Day

An interactive HTML & CSS project created to celebrate **World Environment Day**.  
This webpage showcases eco-friendly slogans that inspire reducing plastic use and promoting greenery.  
Each slogan card reveals its explanation when hovered over, making the experience engaging and dynamic.

---

## ✨ Features
- 📌 Clean and semantic HTML structure  
- 🎨 Modern CSS styling with hover animations  
- 🌱 Eco-friendly color palette (greens & earthy tones)  
- 🖼️ Interactive slogan cards (heading visible, explanation appears on hover)  
- 💡 Motivational quotes for sustainability
